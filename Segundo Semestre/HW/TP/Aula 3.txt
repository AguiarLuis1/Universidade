API Arduino
pinMode(); - serve para caraterizar o pin como entrada ou saída
digitalWrite(); - permite escrever uma tensão num pin digital(HIGH OU LOW)
digitalRead(); - permitie ler a tensão digital num pin digital(HIGH OU LOW)
analogRead(); - lê tensão analógica(continua) que se encontra num pin analógico
analogWrite(); - escreve tensão analógica(continua) que se encontra num pin DIGITAL
delay(); - aguarda o tempo em milissegundos indicado
millis(); - o tempo em milissegundos desde que se ligou

Concepts Input vs Output
Reference from the perspective of the MCU
Inputs: Is a signal information going into time board
Outputs: Is any signal exiting the board

Inputs	|	Output
Sensor		LED
Button		Motor
Potentiometer	7-seguement LED
(...)		(...)

HIGH
Pin is input->digitalRead(pin)		Pin is output->digitalWrite(pin,_)
High if:				High:
	V>=3V(Arduino 5V)		5V(Arduino 5V)
	V>=2V(Arduino 3,3V)		3,3V(Arduino 3,3V)

LOW
Pin is input->digitalRead(pin)		Pin is output
Low if:					Low=0V
	V<3V(Arduino 5V)
	V<2V(Arduino 3,3V)
